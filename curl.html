cURL help

Verbose: the whole page

curl -v http://www.msn.com

Just the headers

curl -I http://www.google.com

Post data

curl -d “string” http://myform.html curl -d @myfile http://myform.html curl -d ‘{ “name” : “rex” }’ http://myform.html

Put data

curl -T http://mysite.html

Get ASCII file

curl “ftp://mysite.com/file.txt;type=A”

Get binary file

curl “ftp://mysite.com/file.bin;type=I”

Get directory

curl “ftp://mysite.com/dir;type=D”
