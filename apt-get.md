title: apt-get 
lang: en
---

***

##### Restart server
```
    sudo shutdown -r now	   
```

***

##### Fetch list of available updates
```
    sudo apt-get update        
```

***

##### Strictly upgrade the current packages
```
    sudo apt-get upgrade       
```

***

##### Install updates (new ones)
```
    sudo apt-get dist-upgrade  
```
